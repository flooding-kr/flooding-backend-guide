# 도메인 용어

혼잡한 단어 사용을 막기 위해서 몇가지 동작들을 분류하여 제한하고자 합니다.

| 수행되는 로직 | 단어     |
| ------------- | -------- |
| 수정          | Update   |
| 조회          | Fetch    |
| 삭제          | Remove   |
| 생성          | Create   |
| 로그인        | SignIn   |
| 로그아웃      | SignOut  |
| 회원가입      | SignUp   |
| 인증          | Verify   |
| 재발급        | Reissue  |
| 취소          | Cancel   |
| 예약          | Reserve  |
| 탈퇴          | Withdraw |
